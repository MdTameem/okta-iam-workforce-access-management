# Workforce Identity & Access Management using Okta

This project demonstrates a real-world implementation of Workforce Identity and Access Management (IAM) using Okta Free Trial. The objective is to manage user access securely, enforce authentication controls, and maintain audit-ready logs.

## Key Objectives
- User lifecycle management
- Group-based access control
- Multi-Factor Authentication (MFA)
- Application access governance
- Audit logging and compliance readiness

## Tools Used
- Okta Workforce Identity (Free Trial)
- Okta Verify (MFA)
- SaaS Application (Office 365 / Slack)

## Implementation Steps
1. Created users to simulate employee onboarding
2. Configured security groups for role-based access
3. Assigned enterprise SaaS applications to groups
4. Enabled MFA to strengthen authentication security
5. Reviewed system audit logs for compliance and monitoring
   

## Security & Compliance Alignment
- Least Privilege Access
- Access Governance
- IT Audit Evidence Collection
- SOX / ISO 27001 aligned controls

## Outcome
This project reflects hands-on experience with IAM operations, security controls, and audit-ready access management used in real enterprise environments.
